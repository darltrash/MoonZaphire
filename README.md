# Chateando

Chateando is a gui frontend for mqtt.
This software require:
* [Lua5.1](https://www.lua.org/download.html)
* [Lgi](https://github.com/pavouk/lgi)
* [Lua MQTT](https://github.com/geekscape/mqtt_lua)
* [Mosquitto/MQTT](http://mqtt.org/)
